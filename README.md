# go-elastic-v5

## Installation

```shell
# 1. clone repository
git clone https://github.com/bagus123/go-elastic-v5.git

# 2. downloads all dependencies
go build

# run from binary
./go-elastic-v5

# or run from source
go run main.go

# note
# clean cache go
go clean --modcache

# remove unused module
go mod tidy
```

## manual

```shell
# get library
go get github.com/elastic/go-elasticsearch/v5
```
